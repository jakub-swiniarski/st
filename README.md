# ST - Simple Terminal
This is a fork of [st](https://st.suckless.org/) from [suckless.org](https://suckless.org).

## Prerequisites:
- gcc,
- make,
- xorg-server,
- libx11,
- libxft.

## Changes:
- changed the default font size,
- applied the scrollback patch (Shift+PageUp/PageDown),
- color changes.
